
ウェブサーバーで、単純な計算を行い、ローカルファイルに結果を書き込みます。

ビルド方法:


$ go build .


実行:

$ ./simplewebapp

計算のコマンドは別のターミナルから入力します。
入力は次の形式です。

ID
OP
VAL1
VAL2

・ID は任意の文字列
・OP は `+`, `-`, `*`, `/` のいずれか
・VAL1 と VAL2 は整数

たとえば次のようになります。

CALC_1
+
3
2


別のターミナルでサーバーを起動しておいてから（./simplewebappを実行）、たとえば次のように実行するとサーバを実行したディレクトリの下の results.txt に結果が書き込まれます。


$ curl -X POST localhost:8080 --data 'CALC_2
*
100
3000'
